About

Notes is a simple web application for creating notes.

It was developed by Stas

You can use it to create notes, like them.

Technologies

  • Spring Boot
  • Spring Security
  • Spring Data MongoDB
  • Thymeleaf
  • Bootstrap
  • MongoDB

Features

  • Registration
  • Login
  • Logout
  • Creating notes
  • Editing notes
  • Deleting notes
  • Like notes
  • Viewing notes
  • Viewing notes by user
